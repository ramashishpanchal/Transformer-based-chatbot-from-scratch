# 🤖 Transformer-Based Chatbot (From Scratch)

## 🚀 Overview
This project focuses on building a **chatbot using a Transformer architecture from scratch** to gain a deep understanding of how modern NLP systems work in real-world scenarios.

Instead of relying on prebuilt models, the entire pipeline — from data processing to prediction — was implemented manually using TensorFlow.

---

## 🎯 Objectives
- Understand the **Transformer architecture** in depth  
- Work with **real conversational datasets**  
- Implement **efficient tokenization and decoding strategies**  
- Bridge the gap between theory and real-world NLP systems  

---

## 🧠 Key Features

### 🔹 Dataset
- Used **Convokit library** to extract conversational data  
- Movie dialogue corpus for realistic human-like conversations  

### 🔹 Model Architecture
- Built using **TensorFlow layers**
- Implemented core Transformer components:
  - Multi-Head Attention  
  - Positional Encoding  
  - Encoder-Decoder structure  

### 🔹 Tokenization
- Used **tiktoken library**  
- Avoided manual tokenization  
- More efficient and industry-relevant approach  

### 🔹 Decoding Strategy
- Implemented **Beam Search** instead of Greedy Search  
- Generates more coherent and context-aware responses
